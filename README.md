# Media Lens PH

Media Lens PH is an interactive web application designed to promote media literacy among Filipinos in the age of globalization. The app helps users understand how media influences globalization and teaches them how to identify misinformation and fake news.

## Features

- **Introduction Section**: Overview of media globalization and its impact on the Philippines
- **Quick Facts**: Relevant statistics about media use and misinformation in the Philippines
- **Interactive Quiz**: 5-question quiz "Can You Spot Fake News?" to test media literacy awareness
- **Proposed Solution**: Recommendations for media literacy education in schools and communities
- **Resource Section**: Links to credible sources like UNESCO and Rappler's fact-check portal
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices

## Technologies Used

- HTML5
- CSS3 (with modern styling and animations)
- JavaScript (for interactive quiz functionality)
- Google Fonts (Poppins)
- Unsplash (for hero image)

## How to Use

1. Open `index.html` in a web browser
2. Navigate through the different sections
3. Take the interactive quiz to test your media literacy skills
4. Explore the resources for further learning

## Deployment

### GitHub Pages

1. Create a new repository on GitHub
2. Upload all files (index.html, styles.css, script.js)
3. Go to Settings > Pages
4. Select the main branch as the source
5. Your site will be available at `https://yourusername.github.io/repository-name`

### Replit

1. Create a new Repl and select "HTML/CSS/JS"
2. Upload all files
3. Click "Run" to preview
4. Share the Repl link

## Target Audience

Filipino students, young professionals, and anyone who uses social media as a main source of information.

## Project Information

**Purpose**: Help users understand the role of media in connecting people globally while showing its possible negative impact when misinformation spreads.

**Relevance**: This project demonstrates how media, as a global driver, connects societies and spreads information rapidly, while emphasizing the need for responsible digital citizenship.
